## 為什麼選這些文件

1. **Base model 完全沒見過**：全部是 NYCU 校內私有規章，Qwen2.5-3B 預訓練語料不會包含
2. **事實密集**：充滿具體的學分數、金額、日期、條文號碼，base model 只能瞎猜
3. **主題多元**：涵蓋學術制度、校園生活、獎學金、場地管理等 7 種不同主題，讓 RAG 檢索時自然產生跨主題雜訊
4. **繁體中文**：全部為台灣正體中文公文格式，展示 finetune 對在地用語的適應效果
5. **體量適中**：~430 chunks 足以產出 100+ QA pairs，同時不會讓 Guru 管線跑太久
